Navigation Bar - Featured Item!

This navigation bar makes a statement, and is rather simple, yet powerful looking. It's a perfect navigation bar for a website used as a portfolio, as it gives all relevant information. With this navigation bar, it also makes it very easy to see which page of the website you are on. To make this navigation section (nav) would contain a an unordered list (ul, li). Each list item contains a link (a) to the correct page.

Footer

This footer is larger than most, but it is easy to navigate. It shows the most pertinent information for contact and location right away, along with a map and links to social media for the company. There's also a button to take you right to the top of the page! This footer (footer) would have 3 icons (img) with text (p) in their own sections (div, p, address). Under that, another section (section, div) with text (p) and icons (img), and a map (api documentation, address).

Welcoming "Subscribe" box

This is a "subscribe" box that opens when you first visit the homepage. It allows the user to subscribe right away, and often get a coupon when they first visit a store's online site, or (as in this case) get a newsletter from the website! For building, it would be built similar to a modal. However, there would be no ".click" action to open it, and it would instead be opened immediately upon the open of the website.

Call to Action

A call to action invites the user to continue on a path, and typically is led into gently. The goal of the template here is to book a client, and it leads the customer to click to get more information on how to learn more about the design/development business before contacting them about a quote. It would be built like any other button, and link to an "about" page.

Working Contact Form

This is a working contact form with a message space. When a user fills in the form and submits it, it sends a message to an email. It would have the basic outline of a form, with not much additional - however, it would have something in the jquery/javascript files that would send anything in the message to an email address.
